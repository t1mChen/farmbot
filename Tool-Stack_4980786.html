HarvestX : Tool Stack

Created by Xinxin Guo, last modified by Mingyu Li on Oct 20, 2023

Type

Tool

Description

IDE

Visual Studio Code

 

  • feature-rich code editor

  • powerful extensions and plugins support different languages, frameworks and tools.

Agile Tool

 

Jira

  • Project management and issue tracking software

  • serves as a dynamic model of current development progress

  • Jira supports Agile methodologies and offers more robust features compared to other project management tools like Trello.

Communication

Slack

 

Zoom

  • Team internal communication platforms.

  • provides more flexibilities to development.

  • where we inspect progress, provide feedbacks, resolve issues, set todos and make decisions.

Designing

Motivational Model: eResearch

Draw.io

Figma

  • motivational modelling editor tool.

  • Diagramming application that contains useful tools for drawing UML diagrams collaboratively.

  • UI design platform with various features where team can design collaboratively.

Documentation

Confluence

  • Documentation and collaboration platform.

  • Transparency and consistency: process and work visible to every member.

  • All information regarding project recorded here.

Tech Stack

Frontend: TypeScript with React

 

Backend: Ruby on Rails

 

Version Control and Collaboration: GitHub

  • Basic technology for developing & modifying current Farmbot Web App system.

Deployment

Server Hosting: Google Cloud Platform

  • GCP provides reliable and scalable server hosting solutions. It boasts a global network infrastructure, ensuring optimal performance and reduced latency.

Retrospective

Miro

  • A visual workspace separate from confluence for team to do retrospective together, providing more flexibilities.

AI photo generator

Canva

  • AI tool that can generate photos according to text descriptions, which is suitable for our project.
