HarvestX : Sequence diagrams

Created by Mingyu Li, last modified on Oct 28, 2023

Sequence diagrams shows a visual representation of how the components of FarmBot system interact and communicate with each other to achieve certain aims. Additionally, it shows all possible outcomes output from the system.

In this section, sequence diagrams serves to illustrate the dynamic behaviour of the system in response to different user actions or external events. Each of them is aligned with a particular use case, clearly demonstrating all possible steps taken by user, flows inside the system and responses returned by it.
