Created by Mingyu Li, last modified on Sep 24, 2023

This sequence diagram shows all possible actions that user can take with the panel of ‘MOVE’ tab in demo mode (shown below), which are move using direction buttons, take photos with camera buttons and reset with reset buttons. It clearly depicts the behaviour of the system in response to different kinds of these events, ensuring a shared understanding between different stakeholders.
