 Problem Statement

In this final sprint, we are embarking on the culmination of our project journey. This sprint holds paramount significance as it represents the last leg of our project's development. Our primary objectives in this sprint are to conclude all remaining tasks, ensure the seamless integration of features, and meticulously prepare for the project's completion.

The following tasks are proposed by our team members, which is expected to be carried out by the end of the sprint.

  • Modify user story base on client's feedback.

  • Overall polish the entire system, conduct performance testing and optimization

  • Finish all user stories

  • Finish all code review and testing (Unit, user experience, integration etc.)

  • Review final product with client

  • Thorough documentation of the project's architecture, codebase, and user guides will be created or updated.

  • Towards the end of this sprint, we will initiate the project closure process, which includes formalizing project documentation, archiving project assets, and conducting a retrospective to capture lessons learned.

 Scope
